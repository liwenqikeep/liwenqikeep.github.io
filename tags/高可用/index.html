Redis（三）：集群与高可用

Redis Redis 主从复制 Sentinel

MySQL（六）：主从复制与高可用

数据库 MySQL 主从复制 binlog

前言 主从复制是 MySQL 高可用架构的基石。通过将一台主库的数据实时同步到多个从库，可以实现读写分离（分担查询压力）、数据备份（冗余存储）和故障切换（高可用）。 本文从复制原理出发，逐步深入到复制模式、GTID、常见问题和架构方案。
